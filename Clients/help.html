ความช่วยเหลือ

วิธีการจอง
  1. ไปที่แท็บ "จองห้องปฏิบัติการ"
  2. กรอกข้อมูลการจอง
  3. เลือกวันที่และเวลา
  4. ยืนยันการจอง
การเขียนโค้ด
  • ใช้ Python 3.7+
  • มี Library robot สำหรับควบคุมหุ่นยนต์
  • ฟังก์ชันหลัก: forward(), backward(), turn_left(), turn_right()
  • เซนเซอร์: distance_sensor(), light_sensor()
ข้อควรระวัง
  • ตรวจสอบการเชื่อมต่อก่อนใช้งาน
  • ทดสอบโค้ดในโหมดจำลองก่อน
  • ไม่เกินเวลาที่จองไว้
  • หยุดหุ่นยนต์เมื่อเสร็จสิ้น
เริ่มต้นใช้งาน
ขั้นตอนการใช้งาน
  1. 1เข้าสู่ระบบด้วยบัญชีของคุณ
  2. 2เลือกฟังก์ชันที่ต้องการใช้งาน
  3. 3จองเวลาใช้งานห้องปฏิบัติการ
  4. 4เขียนโค้ดควบคุมหุ่นยนต์
  5. 5รันโค้ดและดูผลลัพธ์
ข้อกำหนดระบบ
  • เว็บเบราว์เซอร์: Chrome, Firefox, Safari, Edge
  • การเชื่อมต่ออินเทอร์เน็ต
  • ความรู้พื้นฐาน Python
  • บัญชีผู้ใช้งานที่ลงทะเบียนแล้ว
คู่มือการควบคุมหุ่นยนต์
ฟังก์ชันการควบคุมพื้นฐาน
ฟังก์ชัน คำอธิบาย ตัวอย่าง
forward(duration) เดินหน้า robot.forward(2)
backward(duration) ถอยหลัง robot.backward(1)
turn_left(duration) เลี้ยวซ้าย robot.turn_left(0.5)
turn_right(duration) เลี้ยวขวา robot.turn_right(0.5)
stop() หยุด robot.stop()
ฟังก์ชันเซนเซอร์
ฟังก์ชัน คำอธิบาย ค่าที่ได้
distance_sensor() วัดระยะทาง เซนติเมตร (0-200)
light_sensor() วัดความเข้มแสง 0-1023 (0=มืด, 1023=สว่าง)
คำถามที่พบบ่อย (FAQ)
คุณสามารถจองห้องปฏิบัติการได้ตั้งแต่ 30 นาที ถึง 2 ชั่วโมงต่อครั้ง และไม่เกิน 4 ชั่วโมงต่อวัน
ได้ คุณสามารถยกเลิกการจองได้ก่อนเวลาเริ่มต้น 1 ชั่วโมง โดยไปที่หน้า "ประวัติการใช้งาน" และคลิกปุ่ม "ยกเลิก"
กรุณาตรวจสอบการเชื่อมต่ออินเทอร์เน็ต และลองรันโค้ดใหม่ หากยังไม่แก้ไข กรุณาติดต่อผู้ดูแลระบบ
ได้ คุณสามารถบันทึกโค้ดเป็นไฟล์ .py ได้โดยคลิกปุ่ม "บันทึกโค้ด" ในหน้า "ควบคุมหุ่นยนต์"
กรุณาตรวจสอบการเชื่อมต่อกล้อง และลองรีเฟรชหน้าเว็บ หากยังไม่แก้ไข กรุณาติดต่อผู้ดูแลระบบ
